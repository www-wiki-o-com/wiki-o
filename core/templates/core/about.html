About Wiki-O

Wiki-O is a platform for users to express their opinions and learn about other opinions, all while avoiding argumentments. This is achieved by cutting of direct communication between the users and instead having them visually share the set of evidence and theories their opinion is based on.

This website provides a comprehensive set of tools for visualizing opinions such as venn-diagrams, histograms, and pie charts. In this way for any given topic we can quickly and efficently interpret opinions as one-sided, contraversial, baseless, and so on.

Wiki-O Trivia
  • Wiki-O was named after Riki-Oh , the famous eastern philosopher that left no argument unsettled.
  • Riki-Oh was the man.
