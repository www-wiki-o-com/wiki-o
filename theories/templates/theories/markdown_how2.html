  How To Compose the Statement

The evidence is verifiable if and only if anyone with an unlimited amount of resources could confirm the correctness of the statment. For example, the statement "Trump is stupid" is not verifialbe but the statment "Trump claims he is a genius" is verifiable.

We encourage the user to include references that will allow other users to help verify the statement.

  Is this Verifiable?

Statements must be composed without implication. For example the statement "The idiot Trump claims he is a genius" makes the additional implication that Trump is an idiot. It should instead be composed as "Trump claims he is a genius".

  How To Format The Text

The deail section can be composed with Markdown. See the markdown guide for some basic and extended syntax.

  How To Create Links

A link is created with round brackets preceeded by a set of square brackets to provide a discription of the link.

For example: [Wiki-O](http://www.wiki-o.com) will render as Wiki-O.

  How To Create a Bibliography

To create a bibliography, use the link syntax within Markdown: "[#](tags)" inline with your text followed by "[tag]: The bib text." and for bib items that do not appear in the text add them to the bottom surround the bibliography in bib tags.

Example:
To create a bibliography, use the link syntax [#](bib_one, bib2).
<bib>
  [bib_one]: Bib item one.
  [bib2]: Bib item two.
  [wiki_o]: An un-referenced item to [Wiki-O](http://www.wiki-o.com)
</bib>
Renders as:
To create a bibliography, use the link syntax [1,2].

  1. Bib item one.

  2. Bib item two.

  3. An un-referenced item to Wiki-O.
